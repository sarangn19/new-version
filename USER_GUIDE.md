## Study Tools

### Pomodoro Timer

The built-in Pomodoro timer helps you maintain focus and take regular breaks.

#### How to Use:
1. **Access the Timer**
   - Available on most study pages
   - Click the timer icon in the top navigation
   - Or use the dedicated study tools page

2. **Customize Settings**
   - Work Duration: Default 25 minutes
   - Short Break: Default 5 minutes
   - Long Break: Default 15 minutes
   - Sessions until Long Break: Default 4

3. **Start a Session**
   - Click "Start" to begin your work session
   - The timer counts down with visual progress
   - Audio notification when time is up
   - Automatic break timer starts

4. **Track Your Sessions**
   - View completed sessions in your analytics
   - See total focus time per day/week
   - Monitor productivity patterns

### Study Streak Tracker

Maintain motivation with daily and weekly study streaks.

#### Features:
- **Daily Streaks**: Consecutive days of study activity
- **Weekly Goals**: Maintain consistent weekly study time
- **Milestone Celebrations**: Special notifications for achievements
- **Streak Recovery**: Grace period for missed days
- **Motivational Messages**: Encouraging feedback based on progress

#### Building Your Streak:
1. Study for at least 15 minutes daily to maintain your streak
2. Use any study feature (MCQ, flashcards, notes, timer)
3. Check your streak status on the dashboard
4. Aim for weekly milestones (7, 14, 30 days)

### Goal Management

Set and track SMART (Specific, Measurable, Achievable, Relevant, Time-bound) goals.

#### Goal Types:
- **Daily Goals**: Study time, questions answered, topics covered
- **Weekly Goals**: Broader objectives and milestones
- **Monthly Goals**: Long-term preparation targets
- **Subject-Specific**: Goals for individual subjects
- **Custom Goals**: Create your own goal types

#### Creating Goals:
1. Navigate to Goals section in your profile
2. Click "Create New Goal"
3. Choose goal type and timeframe
4. Set specific targets (time, questions, topics)
5. Add deadline and priority level
6. Save and start tracking

#### Tracking Progress:
- Visual progress bars show completion percentage
- Daily/weekly check-ins update automatically
- Notifications for approaching deadlines
- Achievement celebrations when goals are met

### Spaced Repetition System

Optimize your review schedule for better long-term retention.

#### How It Works:
- Algorithm determines optimal review timing
- Difficult topics appear more frequently
- Mastered content has longer intervals
- Performance-based adjustments
- Personalized to your learning patterns

#### Using Spaced Repetition:

## Troubleshooting

### Common Issues and Solutions

#### 1. Data Not Saving
**Problem**: Changes to profile or study data aren't being saved.
**Solutions**:
- Check your browser's localStorage settings
- Ensure you have sufficient storage space
- Try refreshing the page and re-entering data
- Clear browser cache and try again
- Export your data as backup before troubleshooting

#### 2. Performance Issues
**Problem**: The platform is running slowly or freezing.
**Solutions**:
- Close unnecessary browser tabs
- Clear browser cache and cookies
- Disable browser extensions temporarily
- Check your internet connection speed
- Try using a different browser

#### 3. Theme Not Applying
**Problem**: Theme changes aren't taking effect.
**Solutions**:
- Refresh the page after changing themes
- Check if browser supports CSS custom properties
- Clear browser cache and reload
- Try switching to a different theme first
- Ensure JavaScript is enabled in your browser

#### 4. Notifications Not Working
**Problem**: Not receiving study reminders or notifications.
**Solutions**:
- Check browser notification permissions
- Verify notification settings in your profile
- Ensure the browser tab remains open
- Check if "Do Not Disturb" mode is enabled
- Try enabling notifications in browser settings

#### 5. Import/Export Issues
**Problem**: Data import or export is failing.
**Solutions**:
- Check file format compatibility
- Ensure file size is within limits
- Verify file isn't corrupted
- Try exporting smaller data sets
- Use different file format if available
