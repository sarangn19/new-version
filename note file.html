Indian Presidential Election

GS1 Polity Last updated: 22 Oct 2025 Source: The Hindu, MEA, Vision

Core Idea

The President of India is elected indirectly by an Electoral College of MPs and MLAs through a single transferable vote using proportional representation.

Example: In 2017, Ram Nath Kovind won with ~65% of weighted votes.

1. Definition / Background

Note content will appear here

2. Key Features / Components

  1. Electoral College: elected MPs + elected MLAs (incl. UTs with legislatures).
  2. Value of vote: MLA vote value varies by state; MP vote value = total MLA value / total MPs.
  3. System: Proportional representation with single transferable vote, secret ballot.
  4. Nomination: Citizen of India, 35+ years; 50 proposers and 50 seconders; deposit.

3. Significance / Importance

  • Ensures federal balance and broad-based support.
  • Indirect election reduces populism; emphasizes federal character.

4. Issues / Debates

  • Unequal vote value; smaller states over-represented.
  • Non-partisanship vs political realities in campaigns.

5. Government Initiatives / Case Studies

  • 2021 election: Kovind polled 702,044 valid votes (65.65%).

6. Way Forward

  • Periodic review of value-of-vote formula with updated population figures.
  • Greater transparency in preference transfers.
